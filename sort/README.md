####Stable Sort algorithm:

Maintain the relative order of the records with equal keys.

Stable sort:

- Merge sort
- Insertion sort
- Bubble sort
- Bucket sort

Unstable:

- Selection sort
- heap sort
- quick sort (typical in-place sort is not stable, stable versions exist)
